# Vociferous GUI - Keyboard Shortcuts

This document lists all available keyboard shortcuts in the Vociferous GUI application.

## Global Shortcuts

These shortcuts work throughout the application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+,` | Open Settings | Navigate to the Settings screen |
| `Esc` | Cancel/Close | Close navigation drawer or cancel current operation |

## Home Screen Shortcuts

These shortcuts only work on the Home screen:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+O` | Browse Files | Open file browser to select an audio file |
| `Ctrl+T` | Start Transcription | Begin transcribing the selected audio file |
| `Ctrl+S` | Save Transcript | Save the transcript to a text file |

## Tips

- All shortcuts use the `Ctrl` key (or `Cmd` on macOS)
- Shortcuts are case-insensitive
- You can also use drag-and-drop to load audio files on the Home screen
- Hover over buttons to see tooltips with shortcut hints

## Accessibility

The keyboard shortcuts are designed to make the application more accessible and efficient:

- **Power users** can navigate and perform actions without using the mouse
- **Screen reader users** can rely on keyboard navigation
- **Motor impairment** users can use keyboard-only navigation

## Future Enhancements

Planned keyboard shortcuts for future releases:

- `Ctrl+N` - Start new transcription (clear current)
- `Ctrl+E` - Export transcript in different formats
- `Ctrl+H` - Open transcription history
- `Ctrl+P` - Toggle audio playback
- `Space` - Play/pause audio (when playback is implemented)
